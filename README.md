## Myridius Theme V1

HubSpot CMS theme containing:

- `templates/page.html`: assembles the page from modules
- `modules/*`: reusable sections (each includes `module.html`, `module.css`, `module.js`, `fields.json`)

Upload the `Myridius Theme V1` folder as a HubSpot theme, then create a page using the `templates/page.html` template.
